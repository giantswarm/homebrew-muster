# Homebrew Muster

Homebrew tap for [Muster](https://github.com/giantswarm/muster) - Universal Control Plane for AI Agents.

## Installation

```bash
brew tap giantswarm/muster
brew install muster
```

## Upgrade

```bash
brew upgrade muster
```

## About Muster

Muster is a **universal control plane** built on the **Model Context Protocol (MCP)** that solves the MCP server management problem for platform engineers and AI agents.

- **Intelligent MCP Aggregation**: Manage all your MCP servers from a single interface
- **Dynamic Tool Discovery**: Let AI agents discover and use tools on-demand
- **Workflow Automation**: Persist discovered solutions as deterministic workflows
- **ServiceClass Management**: Handle prerequisites automatically

For more information, visit the [Muster repository](https://github.com/giantswarm/muster).
